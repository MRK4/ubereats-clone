
# UberEats Clone

Simple project in order to improve my React skills

## Features

Currently in development

## Tech Stack

- React
- TypeScript
- Tailwind CSS

## How to use

1. Clone the repo

``` bash
git clone https://github.com/MRK4/ubereats-clone
```

2. Install dependencies

``` bash
yarn install
```

3. Start the dev server

``` bash
yarn start
```
